ASTERION Events

Управление мероприятиями и наградами для Discord-сервера. Создавайте ивенты, напоминания и автоматические роли для участников.

Основные команды

Просмотр и участие

/events
Список предстоящих ивентов с отметками времени
/eventhelp
Краткая справка по возможностям и ссылкам на инструкции
/dashboard
Личный кабинет ивентолога: ваши ивенты, статистика, дедлайны
Для ивентологов

Создание и управление

/event title:"..." start:"дд.мм.гггг чч:мм"
Создать новое мероприятие (название, описание, время, призы)
/create_template name:"..."
Сохранить шаблон ивента для повторного использования
/use_template name:"..."
Запустить ивент по существующему шаблону
/list_templates
Показать все доступные шаблоны
/delete_template name:"..."
Удалить шаблон (по имени)
/reschedule_event id:... new_time:"..."
Перенести ивент на новую дату и время
/cancel_event id:... reason:"..."
Отменить своё мероприятие с уведомлением участников
/emergency_cancel id:... reason:"..."
Срочная отмена с пометкой критической причины
/request_backup event_id:...
Запросить подмену организатора
Админские команды

Контроль и аналитика

/admin_events status:"..."
Общий список ивентов с фильтрами
/force_cancel id:... reason:"..."
Принудительная отмена любого ивента
/assign_backup event_id:... organizer:@...
Назначить нового организатора вместо текущего
/admin_report period:"..."
Отчёт по активности ивентологов
/award_prizes event_id:...
Автовыдача призов победителям ивента
Возможности

Что умеет AsterionEvents

  • 📋 Создание мероприятий — полноценная форма с описанием, датой, призами
  • 🔔 Автоматические напоминания — уведомления за 24ч, 1ч, 15мин до старта
  • 🏆 Роли участников — автоматическая выдача ролей победителям
  • 📊 Шаблоны ивентов — сохраняйте настройки для повторяющихся мероприятий
  • ⏰ Перенос и отмена — гибкое управление расписанием
  • 📈 Аналитика — статистика по ивентологам и мероприятиям
